now.prev = begin.prev = nullptr -----> begin присваивается now.next {=a.next=nullptr}, т.е. обнуляется;

now.next = begin.next = nullptr -----> end присваивается now.prev {=a.prev=nullptr}, т.е. обнуляется;

Указатели соседей не изменялись и являются нулевыми;

[a, b]
begin = a
end = b
a.next = b
b.prev = a		
a.prev = b.next = nullptr

1) [b] - В итоге указатель начала и указатель конца равны и указывают на B(2ой элемент). Указатели соседей для B должны быть нулевыми:
now = begin = a

now.prev = a.prev = nullptr -----> begin присваивается now.next{=a.next=b}, т.е. указатель начала указывает на B;

now.next = a.next = b -----> now.next.prev присваивается now.prev, т.е. {now.next.prev=(a.next).prev=} b.prev присвается a.prev{=nullptr}, т.е. обнуляется;

Указатель на следующий для В не изменялся и остается нулевым;

2) [a] - В итоге указатель начала и указатель конца равны и указывают на A(1ый элемент). Указатели соседей для A должны быть нулевыми.
now = {(begin).next=a.next} = b

now.prev = b.prev = a -----> now.prev.next присваивается now.next, т.е. {now.prev.next=(b.prev).next=} a.next присваивается b.next{=nullptr}, т.e. обнуляется;

now.next = b.next = nullptr -----> end присваивается now.prev{=b.prev=}), т.е. указатель конца указывает на A;

Указатель на предыдущий для А не изменился и остается нулевым;

1) [b, c] - В итоге указатель начала должен указывать на B(2ой элемент), указатель на предыдущий для B должен обнулиться. 
Указатели конца и соседей для С(3ьего элемента) не изменяются:
now = begin = a

now.prev = a.prev = nullptr -----> begin присваивается now.next{=a.next=b}, т.е. указатель начала указывает на B;

now.next = a.next = b -----> now.next.prev присваивается now.prev, т.е. {now.next.prev=(a.next).prev=} b.prev присвается a.prev{=nullptr}, т.е. обнуляется;

Никакого взаимодействия с С;

2)[a, c] - В итоге указатель следующего для А(1ого элемента) должен указывать на С(3ий элемент), указатель предыдущего для С должен указывать на А. 
Начало и конец остаются неизменными:
now = {(begin).next=a.next} = b

now.prev = b.prev = a -----> now.prev.next присваивает now.next, т.е. {now.prev.next=(b.prev).next=} a.next присваивается b.next{=c}, т.e. следующий для А это С;

now.next = b.next = c -----> now.next.prev присваивается now.prev, т.е. {now.next.prev=(b.next).prev=} c.prev присвается b.prev{=a}, т.е. предыдущий для С это А;

Никакого взаимодействия с началом и концом;

3)[a, b] - В итоге указатель конца должен указывать на B(2ой элемент), указатель на следующий для B должен обнулиться. 
Указатели начала и соседей для А(1ого элемента) не меняются:
now = {((begin).next).next=(a.next).next=b.next} = c

now.prev = c.prev = b -----> now.prev.next присваивает now.next, т.е. {now.prev.next=(c.prev).next=} b.next присваивается c.next{=nullptr}, т.e. обнуляется;

now.next = c.next = nullptr -----> end присваивается now.prev{=c.prev=b}, т.е. указатель конца указывает на B;

Никакого взаимодействия с А;
